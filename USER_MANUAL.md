# Dream2Play AI - User Manual

## Introduction

Dream2Play AI allows users to transform dream descriptions into playable games using Artificial Intelligence and procedural game generation.

---

## Starting the Application

### Frontend

```bash
cd client
npm run dev
```

### Backend

```bash
cd server
npm run dev
```

---

## How to Use

### Step 1: Enter Your Dream

Navigate to the home page.

Enter a detailed dream description.

Example:

```text
I was flying above a futuristic city while dragons chased me.
```

---

### Step 2: Generate Game

Click the **Generate Game** button.

The AI will:

* Analyze the dream
* Identify characters
* Generate enemies
* Create objectives
* Build a game blueprint

---

### Step 3: Review Generated Blueprint

The system displays:

* Hero
* World
* Enemy
* Boss
* Objective
* Difficulty

---

### Step 4: Play the Game

Click **Play Game**.

The generated game launches inside the browser.

---

## Tips for Better Results

* Use detailed dream descriptions.
* Include locations, characters, and events.
* Describe emotions and actions.

Example:

```text
I was a wizard on Mars fighting giant robots while protecting a colony.
```

## Troubleshooting

### Game Not Generating

* Verify internet connection.
* Check API configuration.
* Ensure OpenAI API key is valid.

### Backend Not Running

```bash
npm install
npm run dev
```

### Frontend Not Loading

