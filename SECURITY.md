# Security Policy

## Supported Versions

Currently, security support is active for the following versions:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not disclose it publicly. Report it directly to the maintainers:

- **Email**: security@dream2play.ai

We take security issues seriously and will respond to reports within 48 hours, providing a resolution path as quickly as possible.
